# LF RFID key file format

## Example
```
Filetype: Flipper RFID key
Version: 1
Key type: EM4100
Data: 01 23 45 67 89
```
## Description

Filename extension: `.rfid`

The file stores single RFID key of type defined by `Key type` parameter

### Version history

1. Initial version.

### Format fields

|Name|Description|
|-|-|
|Key type|Key protocol type|
|Data|Key data (HEX values)|
